# Sonic-Storm
64-bit VST2 plugin that virtualizes 7.1 audio over speakers and headphones.  A retro throwback to popular 1990's "3D sound" technology.  Sonic Storm HP uses head and ear simulation combined with an Ambisonic style soundfield to virtualize 7.1 audio over headphones.  Both work with Equalizer APO as virtual surround sound solutions.

I coded this with Anthropic Fable, and tested it myself using various kinds of content.  It's one of the best "3D sound" simulations I've heard, and can be customized moreso to the user than typical solutions.  CPU usage should be very low on modern PC's, no more than typical solutions, and shouldn't have the kinds of noise and distortion you can hear with convolution based solutions.
